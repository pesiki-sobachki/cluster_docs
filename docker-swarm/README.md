# Развертывание Docker Swarm

Docker Swarm объединяет несколько серверов в единый виртуальный кластер для управления контейнерами.

## 1. Требования
*   Минимум 3 сервера (1 Master, 2 Workers) в одной приватной сети.
*   Открытые порты между серверами (см. `../infra/ports.md`).

## 2. Инициализация Master-ноды
Зайдите на сервер, который будет управляющим (Master), и выполните:

```bash
# Замените IP на реальный приватный IP вашего master-сервера (например, 192.168.x.x)
docker swarm init --advertise-addr <PRIVATE_IP_MASTER>
```

Команда выдаст токен для подключения воркеров. **Сохраните его!
*

## 3. Подключение Worker-нод
На каждом worker-сервере выполните команду, полученную на предыдущем шаге:

```bash
docker swarm join --token <TOKEN> <PRIVATE_IP_MASTER>:2377
```

## 4. Проверка статуса
На master-сервере выполните:

```bash
docker node ls
```
Вы должны увидеть список всех подключенных узлов со статусом `Ready`.
