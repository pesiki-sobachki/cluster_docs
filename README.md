# Cluster Documentation

Этот репозиторий содержит полную документацию по развертыванию и поддержке инфраструктуры кластера.

## Структура

Проект разделен на логические модули:

*   **[Docker Swarm](./docker-swarm/README.md)** — Основа нашего кластера. Инструкции по инициализации, добавлению нод и сетевым настройкам.
*   **[YugabyteDB](./yugabyte/README.md)** — Распределенная база данных. Инструкции по деплою, настройке репликации и работе с UI.

## Быстрый старт

1.  Начните с настройки **Docker Swarm**. Без него кластер не будет работать как единое целое.
2.  После настройки оркестратора переходите к развертыванию **YugabyteDB**.

*Цель документации: позволить любому новому инженеру развернуть кластер с нуля, следуя пошаговым инструкциям.*
